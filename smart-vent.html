SCVAIR Smart Vent

Description

The SCVAIR Smart Vent is a project that is based on Raspberry Pi. It is a vent which allows the user to open and control the vent flaps, and enhance the air flow by using a fan built into the vent. This project has its own dedicated website. Click here to visit it where it will show all its features.
